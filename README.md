# Algoritmos e Lógica de Programação com Python 

Este repositório reúne os projetos práticos desenvolvidos por mim para a disciplina de Lógica de Programação no curso de Engenharia de Software na Uninter.

O objetivo deste trabalho foi aplicar conceitos fundamentais de lógica de programação, estruturas condicionais, laços de repetição, funções e estruturas de dados (listas e dicionários).

---

##  Tecnologias e Conceitos Utilizados
* **Linguagem:** Python 3.x
* **Conceitos:**
  * Estruturas condicionais (`if`, `elif`, `else`)
  * Estruturas de repetição (`while`, `break`, `continue`)
  * Tratamento de exceções (`try`, `except`)
  * Manipulação de Listas e Dicionários (`append`, `copy`, funções de busca)
  * Modularização com Funções

---

##  Descrição dos Projetos Acadêmicos

### 1. Sistema de Cálculo de Plano de Saúde (`questao1_plano_saude.py`)
* **Descrição:** Algoritmo que recebe o valor base de um plano de saúde e a idade do beneficiário, aplicando um percentual de reajuste progressivo com base na faixa etária informada.

### 2. Sistema de Vendas de Pizzaria (`questao2_pizzaria.py`)
* **Descrição:** Um sistema de ponto de venda (PDV) interativo para uma pizzaria. O programa valida o tamanho (P, M, G) e o sabor (Salgada ou Doce) escolhidos, acumula o valor total de múltiplos pedidos e exibe o resumo financeiro ao encerrar.

### 3. Sistema de Vendas e Logística - Madeireira (`questao3_madeireira.py`)
* **Descrição:** Sistema modularizado que gerencia a venda e o transporte de toras de madeira. O programa calcula descontos progressivos por volume de metros cúbicos (m³) e soma o custo fixo do modal de transporte escolhido (Rodoviário, Ferroviário ou Hidroviário).

### 4. Gerenciador de Lista de Contatos (`questao4_sistema_contatos.py`)
* **Descrição:** Um CRUD (Create, Read, Update, Delete) em memória para gerenciamento de contatos. Permite cadastrar contatos com ID gerado automaticamente, realizar consultas por ID ou atividade específica, e remover registros da lista.
